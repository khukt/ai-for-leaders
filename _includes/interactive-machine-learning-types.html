Evaluation Lens
Compare outputs to known answers

Performance is checked against labelled test data and task-specific error metrics.

Best Business Use

Prediction and classification tasks where the outcome is already defined clearly enough to label examples.

Main Data Need

High-quality labelled data that is relevant to the real task and population.

Governance Watchout

Bad labels, skewed training data, and hidden outcome bias can make a model look precise while still producing harmful decisions.

Leadership Question

Do we trust the labels and outcome definitions enough for people to rely on this model in real decisions?

Key idea: the learning type changes the evidence burden. Different approaches need different data, different evaluation methods, and different oversight.
