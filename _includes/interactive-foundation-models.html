Keyword Prompting

Giving instructions to the pretrained model without changing the model itself.

What changes

The instructions and surrounding workflow change, but the base model stays the same.

What stays the same

The pretrained model remains the same general-purpose model trained beforehand.

Decision hint

Use this when you mainly need clearer instructions, not new company knowledge or changed model behaviour.

Rule of thumb: if you are only changing the instructions, you are still using the same pretrained model.
