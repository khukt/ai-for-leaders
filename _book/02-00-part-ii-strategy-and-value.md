# Part II — AI Strategy and Value Creation

This part answers four practical leadership questions:

- where does AI create real value for the organisation?
- which capabilities should be built, bought, or developed through partners?
- what data conditions make those bets reliable and defensible?
- how should investment be sequenced so that pilots become durable capability?

The argument of this section is deliberate. Value comes first. Sourcing and data then determine whether that value can be delivered with acceptable cost, control, and trust. The roadmap chapter brings those choices together into a sequence for investment, adoption, and scale.

Leaders should read these chapters as one strategic arc rather than as separate topics. A weak value thesis makes sourcing irrelevant. Poor data quality weakens even a promising use case. A good pilot without a roadmap becomes a local experiment instead of an organisational capability.
