# AI for Management Leaders (Open-Source Book)

This repository is a **production-ready Jekyll + Just the Docs** book site for GitHub Pages.

## Quick start (GitHub Pages)
1. Push this repository to GitHub.
2. In GitHub: Settings → Pages → Source = **GitHub Actions** (recommended) or **Deploy from branch**.
3. If your site is `https://USER.github.io/REPO/`, set `baseurl: "/REPO"` in `_config.yml`.

## Local preview
```bash
bundle install
bundle exec jekyll serve
```
Open the URL shown in the terminal.

## Book content
Chapters live in the `_book/` collection. Add/edit chapters there.
